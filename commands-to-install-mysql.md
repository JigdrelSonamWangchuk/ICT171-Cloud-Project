# Commands to Install MySQL

## Update Packages
```bash
sudo apt update
```

## Install MySQL
```bash
sudo apt install mysql-server
```

## Ro check if it is running
```bash
sudo systemctl status mysql
```
## Log Into MySQL
```bash
 sudo mysql
```

 ## Create Database
```bash
 CREATE DATABASE clothing_store;
```

 ## Check:
```bash
 SHOW DATABASES;
```

 ## Exit
```bash
 EXIT;
```
