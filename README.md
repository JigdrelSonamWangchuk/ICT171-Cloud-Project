# ICT171-Cloud-Project
## Cloud Server Project Documentation
## Project Overview
- This project demonstartes the deployment of a cloud-hosted personal business clothing website using Microsoft Azure Infrastructure as a service (IaaS). The server is hosted on a Ubuntu Linux virtual machine and is configured with a custom domain name and SSL certificate to provide secure HTTPS access. 

My documnetation contains these files as follows:
1. Cloud Server Documnetation
2. Commands to Access Virual Machine
3. Commands to configure Ubuntu Firewall (UFW)
4. Commands to create proper nginx server block
5. Commands to install SSL
6. Commands to install MySQL data management system
7. Screenshots
